- SPACE: stop / play

# Development

- Place `sunvox.wasm` in the `dist` folder
  - [Official download of SunVox library](https://warmplace.ru/soft/sunvox/sunvox_lib.php)
- `npm i`
- `npm run dev`
- Access http://localhost:5124/dist/index.html
